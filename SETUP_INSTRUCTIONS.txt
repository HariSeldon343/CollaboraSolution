================================================================================
TROUBLESHOOTING
================================================================================

ERROR: Database connection failed
----------------------------------
1. Check that MySQL/MariaDB is running in XAMPP Control Panel
2. Verify database settings in /config.php:
   - DB_HOST should be 'localhost'
   - DB_USER should be 'root'
   - DB_PASS should be '' (empty for XAMPP default)

ERROR: 500 Internal Server Error on API endpoints
--------------------------------------------------
1. Run the database initialization script first
2. Check that all required tables exist
3. Verify that you're logged in (session must be active)

ERROR: Tables not found
------------------------
1. Run: http://localhost/CollaboraNexio/setup/init_database.php
2. This will create all required tables automatically

ERROR: CSRF token validation failed
------------------------------------
1. Clear your browser cookies
2. Log out and log in again
3. Make sure cookies are enabled in your browser

================================================================================
API ENDPOINTS
================================================================================

System Status (No auth required):
- GET /api/system/status.php
  Returns system health and component status

User Management (Requires authentication):
- GET /api/users/list.php?page=1&search=
  Get paginated list of users

- GET /api/users/tenants.php
  Get list of available tenants

- POST /api/users/create.php
  Create a new user

- POST /api/users/update.php
  Update existing user

- DELETE /api/users/delete.php
  Delete a user

All authenticated endpoints require:
- Active session (logged in)
- CSRF token in header: X-CSRF-Token
